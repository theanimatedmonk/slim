# Slim SVG

An Android-focused SVG optimisation tool. Upload your assets — the system optimizes them automatically.

## Features

- **Drag-and-drop upload** — batch SVG uploads with progress tracking
- **Automatic SVGO optimization** — iterative passes until size reduction stabilizes (no manual settings)
- **Complexity analysis** — classifies each asset as simple, moderate, or complex after optimization
- **WebP recommendations** — suggests raster conversion when an SVG stays heavy or contains embedded images
- **WebP conversion** — one-click convert for recommended assets
- **Optimization pass history** — per-asset breakdown of each pass and size reduction
- **ZIP download** — bundle optimized SVGs, WebP files, and a JSON report
